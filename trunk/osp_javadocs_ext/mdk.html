Plugin Creation

Step 1. Create JSP page to be shown.
Step 2. Create the XML descriptor of the page.
Step 3. (Optional)

Create a jsp 'make' page to allow the simulation author to control aspects of the JSP page to be shown.

For example, perhaps you are going to have the JSP provide more information to control characters if the simulation author thinks that that is appropriate. You will need to create a page to give the simulation author that option.
Step 4. (Optional)

If you have done step 3, you will probably need to do step 4. You will need to create a class that takes the settings entered on the 'make' page and addes them to information stored in the customized section. This information will then be pulled out when the page displays itself and control aspects of the display.
