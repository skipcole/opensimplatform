The Guide to Creating USIP OSP Plugins.

This guide will walk you (a developer) through the process of creating a plugin for the USIP OSP. this will allow you to provide people with additional functionality in their simulations.

The Process at a Glance

  • Step 1. Copy Files
    The first thing you will do will be to copy all of the template files into a directory you create for your plugin files.
  • Create the XML definitin file (for example grid_doc_v0_0_0.xml)
  • Install your section on a development environment. (Admin --> Extension Tasks --> Install Simulation Sections)
  • navigate to where you will add the plugin section
  • review and fix if necessary the page creation page
  • Add the section to the sample player (maybe we should discuss how to create minimal sim with minimal player)
  • create running sim and login as player who has the new plugin section
  • review and fix, if necessary, the new plugin section
